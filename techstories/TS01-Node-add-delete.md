# Node behaviour in Network

## Terms
A node is any device of a Player which allows to play the game. The network is the networks of nodes, including that nodes which have special purposes and not directly representing a Player.

## Node adding
If a Player enters the Game - and befor login - a node representing the Player will be added to the network. The network of nodes shall be designed such that a node can be added to the network of nodes If added, the new node shall get all updates of the current state of gaming world


## Node log-out
If deleted meaning the player log out, the last state related to the player associated with the noded shall be saved. After this, the node is deleted from network.
